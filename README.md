# ProtectWorld
Nukkit plugin Protect World


# Commands!

  - /pw help
  - /pw add <level_name>
  - /pw remove <level_name>
  - /pw <1/place> <level_name>
  - /pw <2/break> <level_name>
  - /pw <3/pvp> <level_name>
  - /pw <4/damage> <level_name>
  - /pw <5/flint> <level_name>

### config.json


```sh
{
  "worlds": {
    "world": {
      "place": true,
      "break": true,
      "pvp": true,
      "damage": true,
      "flint_and_steel_use": true
    }
  }
}
```
